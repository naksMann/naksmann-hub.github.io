01 — About

Defensive depth. Offensive precision.

I'm a final-year Computer Systems Engineering student at Tshwane University of Technology (Soshanguve South, 2022–2026) and a Cybersecurity Intern at the 4IR Lab in Soshanguve. My work sits at the intersection of Security Operations (SOC), offensive security, and security engineering, where I design, build, and validate security solutions from the ground up.

I learn by building realistic environments and validating them against real-world threats. In Microsoft Azure, I deployed and intentionally exposed a Windows virtual machine to the public internet to study adversary behavior, capturing more than 2,600 brute-force attempts from a single source within hours. I transformed those observations into actionable detections by developing custom Kusto Query Language (KQL) analytics, writing Sigma detection rules, and implementing SOAR automation with human approval workflows to balance speed with operational control.

On the offensive side, I perform black-box penetration testing to identify, validate, and demonstrate exploitable weaknesses. My projects include chaining unrestricted file upload vulnerabilities into remote code execution (RCE), assessing web applications, and documenting practical remediation strategies that improve an organization's security posture.

My goal is to contribute to high-performing security teams where I can combine offensive testing, threat detection, incident response, and cloud security engineering to help organizations anticipate, detect, and respond to evolving cyber threats. I am currently pursuing graduate opportunities in Penetration Testing, Security Operations (SOC), Detection Engineering, and ICS/OT Cybersecurity.
